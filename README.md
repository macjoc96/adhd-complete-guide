# ADHD: Kompleksowy Przewodnik 360°

**Polska książka o ADHD oparta na najnowszych badaniach naukowych**

## O Projekcie

Ta książka to kompleksowy przewodnik po ADHD (Attention-Deficit/Hyperactivity Disorder), łączący:
- **Naukę**: neurobiologia, psychiatria, psychologia
- **Praktykę**: techniki, strategie, narzędzia
- **Nowoczesność**: biohacking, personalizacja, nootropiki

### Podejście
- Evidence-based (oparte na dowodach naukowych)
- Interdyscyplinarne
- Praktyczne i actionable
- Profesjonalne ale przystępne
