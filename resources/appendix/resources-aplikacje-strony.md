# Appendix: Resources - Aplikacje, Strony, Community

## Aplikacje do Zarządzania ADHD

### Task Management & Productivity

**Todoist**
- Typ: Task manager
- Platformy: iOS, Android, Web, Desktop
- Cena: Free + Premium
- ADHD-friendly features: Projekty, recurring tasks, priorytety, integracje

**Things 3** (Apple only)
- Typ: GTD task manager
- Platformy: iOS, macOS
- Cena: Płatna (osobne zakupy na różne platformy)
- ADHD-friendly: Prosty UI, "Today" view, checklist

**TickTick**
- Typ: Task manager z Pomodoro timer
- Platformy: iOS, Android, Web, Desktop
- Cena: Free + Premium
- ADHD-friendly: Wbudowany Pomodoro, habit tracking, kalendarz

**Forest**
- Typ: Focus timer (gamification)
- Platformy: iOS, Android
- Cena: Płatna (iOS), Free+Premium (Android)
- ADHD-friendly: Gamifikacja skupienia, blokowanie rozpraszających aplikacji

**Focusmate**
- Typ: Virtual body doubling
- Platforma: Web
- Cena: Free tier + Premium
- ADHD-friendly: Scheduled accountability sessions z innymi osobami

**Pomodoro Timer Apps**:
- **Focus To-Do** (Free, iOS/Android/Desktop)
- **Be Focused** (macOS/iOS)
- **Pomotodo** (Cross-platform)

**Time Timer**
- Typ: Visual timer
- Platformy: iOS, Android (+ fizyczne urządzenia)
- Cena: Płatna
- ADHD-friendly: Wizualizacja upływającego czasu

### Notatki & External Brain

**Notion**
- Typ: All-in-one workspace
- Platformy: iOS, Android, Web, Desktop
- Cena: Free + Premium
- ADHD-friendly: Flexible, templates, databases

**Obsidian**
- Typ: Markdown note-taking, knowledge base
- Platformy: Desktop, Mobile
- Cena: Free + Premium (sync)
- ADHD-friendly: Backlinks, graph view, offline-first

**Apple Notes / Google Keep**
- Typ: Quick capture
- ADHD-friendly: Szybki dostęp, synchronizacja

---

### Habit Tracking

**Habitica**
- Typ: Gamified habit tracker
- Platformy: iOS, Android, Web
- Cena: Free + Optional premium
- ADHD-friendly: RPG-style gamification, social features

**Streaks** (iOS)
- Typ: Minimalist habit tracker
- Cena: Płatna
- ADHD-friendly: Prosty UI, widget, max 12 habits

**Done** / **HabitNow**
- Typ: Habit tracking
- Platformy: iOS/Android
- Cena: Free + Premium

---

### Focus & Distraction Blocking

**Freedom**
- Typ: Website/app blocker
- Platformy: iOS, Android, Desktop
- Cena: Subscription
- Features: Scheduled blocks, sync across devices

**Cold Turkey** (Desktop)
- Typ: Hardcore app/website blocker
- Platforma: Windows, macOS
- Cena: Free + Premium

**StayFocusd** (Chrome extension)
- Typ: Website limiter
- Cena: Free

---

### Medication Reminders

**Medisafe**
- Typ: Medication reminder & tracker
- Platformy: iOS, Android
- Cena: Free + Premium
- Features: Reminders, refill alerts, family sharing

**Round Health**
- Typ: Medication tracker
- Platformy: iOS, Android
- Cena: Free + Premium

---

### Sleep & Circadian Rhythm

**Sleep Cycle**
- Typ: Sleep tracking z smart alarm
- Platformy: iOS, Android
- Cena: Free + Premium

**f.lux / Night Shift**
- Typ: Blue light reduction
- Platformy: Desktop / Built-in iOS/macOS
- Cena: Free
